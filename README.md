# Xiaomi Airpurifier 4 Compact to InfluxDB

## Installation

```pip install -r requirements.txt```

## Configuration

Create a .env file and write inside the following variables:

* INFLUXDB_TOKEN
* INFLUXDB_ORG
* INFLUXDB_HOST
* INFLUXDB_DATABASE
* XIAOMI_USER
* XIAOMI_PASSWORD

## Running

```python get_air_quality.py```
